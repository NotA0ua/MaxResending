# MaxResending
## Описание
Телеграм-бот для пересылки сообщений из мессенджера Max.

Отправляет сообщения в определённую телеграм группу из указанных чатов в Max. 

## Установка
- **Необходим uv**
- Настройка .env

## Запуск
```
uv run -m app
```

## Переспективы
- Отправка сообщений по топикам в группе
- Docker

## Послесловие
Создано при помощи библиотеки [PyMax](https://github.com/ink-developer/PyMax).

Автор не преследует цели дискредитировать мессенджер Max. Все действия направлены исключительно на повышение удобства использования.
